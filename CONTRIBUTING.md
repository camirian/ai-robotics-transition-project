# Contributing to AI Robotics Transition Project

Thank you for your interest! Contributions are welcome to enhance the MBSE-to-LLM translator or swarm simulation.

## How to Contribute
1. Fork the repository.
2. Create a branch: `git checkout -b feature/your-feature`.
3. Commit changes: `git commit -m "Add your feature"`.
4. Push to your fork: `git push origin feature/your-feature`.
5. Open a pull request with a clear description.

## Guidelines
- Follow PEP 8 for Python code.
- Include tests in `src/*/tests/`.
- Update documentation in `docs/`.
- Ensure compatibility with Python 3.10 and ROS 2 Humble.

## Issues
Report bugs or suggest features via GitHub Issues, using the provided template.
